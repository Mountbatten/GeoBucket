About GeoBucket

  • Back

GeoBucket is a mapping tool that works by geolocating latitude and longitude as you walk or drive and saves these points to a local database. The app then creates a .wkt file from the stored coordinates and uploads this to the GeoBucket Website

The Start button turns on the GPS if its off and initiates saving of coordinates. Everytime a point is saved the Bucket Entries Captured count is incremented by one. The Stop button turns off GPS and halts saving of coordinates.

Upload pushes your traces to the GeoBucket website to be rendered on a map. Uploads can either be anonymous or authenticated at the Settings option. Uploads are done in batches that can be adjusted from the Settings option.

Settings provides an option to Login to the GeoBucket website for uploading trace points; You can also reset the bucket which deletes the database record of all saved trace points that have already been uploaded and clears the Bucket Entries Uploaded count. It also logs you out of the Geobucket website. Reset willnot delete database entries that have not been uploaded; You can also change the Batch Size for uploads which determines how many trace points will be included in an instance Upload.
